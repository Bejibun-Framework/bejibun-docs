`@bejibun/utils` provides small helpers and facades used throughout the framework — and available to your application code. It's a standalone package you can use in any Bun project.

## Helper functions

```ts
import {
    isEmpty, isNotEmpty, defineValue,
    ask, isCommandExists, isModuleExists
} from "@bejibun/utils";
```

| Function | Description |
| --- | --- |
| `isEmpty(value)` | `true` for empty / nullish values |
| `isNotEmpty(value)` | Inverse of `isEmpty` |
| `defineValue(value, default = null)` | Returns `value` if non-empty, otherwise `default` |
| `ask(question)` | Read a line of input from the terminal |
| `isCommandExists(command)` | Whether a shell command is available |
| `isModuleExists(module)` | Whether a package is installed |

```ts
const port = defineValue(env("APP_PORT"), 3000);
if (isEmpty(body.name)) throw new RuntimeException("Name required.");
```

## Str facade

String transformations:

```ts
import Str from "@bejibun/utils/facades/Str";

Str.toPascalCase("hello world"); // "HelloWorld"
Str.toLowerCase("HELLO");
Str.toUpperCase("hello");
Str.random(16);                  // random string of length 16
Str.ipToFileName("127.0.0.1");   // filesystem-safe form of an IP
```

## Enum facade

Work with TypeScript enums by key or value:

```ts
import Enum from "@bejibun/utils/facades/Enum";
import HttpMethodEnum from "@bejibun/utils/enums/HttpMethodEnum";

Enum.setEnums(HttpMethodEnum).toArray();        // [{ key, value }, ...]
Enum.setEnums(HttpMethodEnum).hasValue("GET");
Enum.setEnums(HttpMethodEnum).getKey("GET");
```

## Luxon facade

A thin re-export of [Luxon](https://moment.github.io/luxon) for date and time handling — the same library the framework uses for model timestamps:

```ts
import Luxon from "@bejibun/utils/facades/Luxon";

const now = Luxon.DateTime.now();
const later = now.plus({ hours: 2 });
```

`Luxon.DateTime`, `Luxon.Duration`, and `Luxon.Interval` are all available.

## Object facade

Serialization and form-data parsing helpers (these power `BaseController.parse()`):

```ts
import BObject from "@bejibun/utils/facades/Object";

BObject.serialize(value);          // normalize a JSON value
BObject.parseFormData(formData);   // FormData -> plain object
```

<Note>
Import the `Object` facade under an alias (e.g. `BObject`) to avoid shadowing the global `Object`.
</Note>

**Bejibun** brings Laravel-style developer experience to the Bun runtime.
It's a batteries-included, MVC, TypeScript-first framework with everything you need to build production APIs -- routing,
controllers, validation, an ORM with soft deletes, migrations, queues, scheduling, WebSockets, caching, storage,
and an `ace` CLI -- without assembling a dozen libraries yourself.

## Quickstart

<Steps>
    <Step title="Install Bun">
        **Linux / MacOS**
        ```bash
        curl -fsSL https://bun.sh/install | bash
        ```

        **Windows**
        ```bash
        powershell -c "irm bun.sh/install.ps1 | iex"
        ```
    </Step>
    <Step title="Create a project">
        ```bash
        bunx @bejibun/cli my-app
        ```
    </Step>
    <Step title="Run the dev server">
        ```bash
        cd my-app
        bun dev
        ```
        Your app is now running at `http://127.0.0.1:3000`.
    </Step>
</Steps>

## Built on a modern stack

| Layer                             | Technology                                            |
| --------------------------------- | ----------------------------------------------------- |
| Runtime, package manager, bundler | [Bun](https://bun.com)                                |
| ORM & query layer                 | [Objection.js](https://vincit.github.io/objection.js) |
| Migrations & seeders              | [Knex.js](https://knexjs.org)                         |
| Validation                        | [Vine.js](https://vinejs.dev)                         |
| Date & time                       | [Luxon](https://moment.github.io/luxon)               |

# Roadmap

Bejibun is continuously evolving to provide a modern, productive, and scalable development experience for TypeScript applications running on Bun.

This roadmap highlights the major areas of investment and the features planned for future releases.

The roadmap serves as a guide rather than a strict commitment. Priorities may change based on community feedback, ecosystem developments, and real-world adoption.

---

# Vision

The long-term goal of Bejibun is to become a complete application framework for Bun.

The framework aims to provide:

```text
Developer Experience
        +
Type Safety
        +
Performance
        +
Scalability
        +
Extensibility
```

while maintaining a consistent and intuitive developer experience.

## Authorization

Authentication identifies users.

Authorization determines what users are allowed to do.

Planned capabilities:

```text

Permissions

Policies

Guards

Resource Authorization
```

This will provide a complete access-control solution.

## Scheduler

Applications often need recurring tasks.

Examples:

- Cleanup jobs
- Reports
- Data synchronization
- Notification delivery

Planned syntax:

```ts
Scheduler.command(
  "report:generate"
).daily();
```

Goal:

```text
Cron-Like Scheduling
Using Framework APIs
```
